# Хороший Сервис (Garaj)

Современный сайт автосервиса на Next.js (App Router) + TypeScript + Tailwind в стиле BMW M.

## Страницы
- `/` — главная
- `/prices` — прайс + калькулятор
- `/book` — запись в сервис
- `/about` — отдельная страница «О нас»
- `/admin-login` — вход в админку
- `/admin` — заявки
- `/admin/dashboard` — отдельный мониторинговый дашборд с почасовыми графиками

## Env
```env
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
ADMIN_PASSWORD=
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

## Запуск
```bash
npm i
npm run dev
```
