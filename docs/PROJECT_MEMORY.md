# Память проекта Garaj (детальное состояние)

## 1) Технологический стек
- Framework: Next.js 14 (App Router), React 18, TypeScript.
- UI: Tailwind CSS.
- Серверные обработчики: Route Handlers в `app/api/*`.
- Хранилище данных: локальный JSON-файл `.data/store.json`.

## 2) Текущая структура и ответственность
- `app/page.tsx`: рендер главной страницы через `HomeContent`.
- `app/components/home-content.tsx`:
  - шапка с якорями (`Прайс`, `О нас`, `Записаться`),
  - слоган: «Твоя скорость — наша точность»,
  - интро,
  - прайс (поиск + фильтр),
  - пустой раздел `О нас` (placeholder),
  - калькулятор с мультивыбором,
  - форма заявки с anti-spam (honeypot), consent-чекбоксом и красивым file-input.
- `data/services.ts`:
  - источник истины для услуг,
  - категория/единица/note,
  - форматирование цен.
- `app/api/telegram/route.ts`:
  - валидация имени/телефона/consent,
  - защита honeypot,
  - отправка в Telegram,
  - запись заявки в `.data/store.json`.
- `app/api/metrics/route.ts`:
  - трекинг пользовательских событий (`transitions`, `formClicks`, `submitClicks`).
- `lib/store.ts`:
  - чтение/запись `.data/store.json`,
  - `submissions`, `metrics`, `metricHistory` (помесячно/посуточно).
- `app/admin/page.tsx`:
  - метрики,
  - таблица заявок,
  - график посещаемости,
  - кнопка выхода.
- `app/admin/metrics-chart.tsx`:
  - фильтруемый график (метрика + период 7/14/30 дней).

## 3) Безопасность и доступ
- Токены Telegram только на сервере (env).
- Админка защищена паролем:
  - `app/admin/layout.tsx` проверяет cookie,
  - логин через `/admin-login`,
  - API входа `/api/admin/login`.
- Выход: `/api/admin/logout`.
- Важная env-переменная: `ADMIN_PASSWORD`.

## 4) События и аналитика
- `transitions` — клики на якоря навигации.
- `formClicks` — взаимодействие с формой.
- `submitClicks` — нажатия отправки.
- История метрик агрегируется по ключу даты `YYYY-MM-DD`.

## 5) UX и визуальная концепция
- Тёмная тема, жёлто-янтарные акценты, «гаражный» и молодёжный стиль.
- Автотематика через слоган, иконки соцсетей, контрастные неоновые карточки и кнопки.
- Калькулятор подготавливает поле «Необходимые работы» и ведёт к форме заявки.

## 6) Ограничения текущей версии
- File upload сейчас демонстрационный: в Telegram/хранилище уходит только имя файла.
- Админка базово защищена паролем, без ролей и MFA.
- Файл `.data/store.json` подходит для MVP, но не для high-load/prod масштаба.
